<h1>WeatherApp</h1>

<p>Simple Weather app written in HTML, CSS, and JavaScript using the APIs for fetching weather and geolocation information.</p>

<h3>Used Technologies</h3>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript</li>
</ul>

<h3>Used APIs</h3>
<ul>
  <li>Open Weather API</li>
 <li>Unsplash API</li>
</ul>
#### Steps to Use:
---

- Download or clone the repository
```
git clone https://github.com/K-umaraswamy/weather-app.git
```
- Go to the directory
- Run the index.html file
- Just type the city name you want to know the weather of, and then click on the search button.
- After that it will automatically display Weather, Humidity, Wind Speed of that city.
